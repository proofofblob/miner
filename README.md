# miner


## 1. install golang

download from https://go.dev/dl/

## 2. clone repo

```shell
git clone https://github.com/proofofblob/miner.git
```

## 3. compile miner

```shell
cd miner

go build .
```


## 4. run miner

```shell 
# windows
./miner.exe

# linux/unix

./miner
```

Then enter the information as prompted
