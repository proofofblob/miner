# miner


## 1. install golang

download from https://go.dev/dl/

## 2. clone repo

git clone
```shell
git clone https://github.com/proofofblob/miner.git
```

or download zip
https://github.com/proofofblob/miner/archive/refs/heads/main.zip

## 3. compile miner

```shell
cd miner

go build .
```


## 4. run miner

```shell 
# windows
./miner.exe

# linux/unix
./miner
```

Then enter the information as prompted
